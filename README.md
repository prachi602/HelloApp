# HelloApp
PROJECT PURPOSE:
Hello App starts with Hello World, progresses to displaying a user name, then names from command-line args and standard input. It then manages names in a collection with list/remove options, refactors into methods and classes, adds persistence across runs, and finally displays names in banner format.

🚀 How to Run HelloApp:

Compile the code  
From the project root, run:
javac -d bin src/HelloApp.java
javac compiles your Java source file.
(-d bin tells it to place the compiled .class files inside the bin directory.
src/HelloApp.java is your source file.)

src/HelloApp.java is your source file.

Run the program  
From the project root, run:
java -cp bin HelloApp
java runs the compiled program.
(-cp bin sets the classpath to the bin directory where compiled files are stored.
HelloApp is the name of your main class.)
